DaShuiGUaiProject
==============
A Blog WebSite based on Asp.net MVC 4

--------------

__Web design:__  
* keenthemes UI framework  

__Infrastructure:__

* Log4Net  
* AutoMapper  
* Castle.Windsor
* MvcPager

* comment components:  

__Features:__
